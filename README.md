# Adhithya Mohan — Living Digital Architecture (v8.5)

This is not a traditional web portfolio. It is a highly optimized, psychologically immersive runtime environment designed to blur the line between software engineering and digital consciousness.

## 🧠 The Philosophy
The goal of this architecture is **Restraint** and **Believability**. 
Rather than overwhelming the visitor with constant visual spectacle, the system is designed to hesitate, reflect, remember, and occasionally embrace complete silence. It is an evolving digital presence that quietly exists on the internet.

## ⚙️ Core Technology Stack
* **Engine:** React 19
* **Physics & Motion:** Framer Motion 12
* **Atmospherics:** Three.js & tsParticles
* **Content Layer:** Sanity CMS
* **Environment Symbiosis:** OpenWeather API (Thiruvananthapuram)

## 🏗️ Architectural Systems
### The SystemOrchestrator
To maintain 60fps cinematic fluidity, high-frequency DOM updates and atmospheric particle physics have been completely decoupled from React's rendering lifecycle. A single, unified `requestAnimationFrame` loop manages all global interactions, preventing layout thrashing and battery drain.

### Dynamic Performance Tiers & Tab Consciousness
The engine actively monitors your device's FPS capability. If it detects severe frame drops, it autonomously degrades to "Static Mode", halting intensive DOM matrix transformations. Furthermore, if the tab loses visibility, the engine enters a deep Sleep State, zeroing out CPU/GPU overhead.

### The Silence Engine
The portfolio is emotionally aware of its own content. When a visitor lingers in deeply reflective sections (like *Digital Scars* or *Timeline*), the architecture voluntarily mutes its background visuals, shrinking the Digital Soul and halting distractions to prioritize emotional intimacy.

### The Digital Subconscious
Underneath the surface runs an ambient thought engine. It remembers recurring visitors, calculates "adrenaline" based on cursor velocity, and occasionally leaks corrupted memory fragments ("Failed Selves") into the interface, creating a subtle layer of digital archaeology.
